# OpenChat SDK

A Flutter+Rust based chat SDK for Android integration.

## Quick Start

```gradle
repositories {
    maven { url 'https://jitpack.io' }
}

dependencies {
    implementation 'com.github.rookie444:openchat-sdk:v1.0.0'
}
```
